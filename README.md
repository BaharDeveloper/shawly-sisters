# Shawly Sisters

Shawly Sisters Wedding Favors — el yapımı düğün hediyeliği ve mum satış sitesi.

## Özellikler

- Ürün listeleme ve sepet
- Kullanıcı kayıt / giriş + şifremi unuttum
- Sipariş tamamlama ve sipariş geçmişi (3 iş günü tahmini teslim)
- WhatsApp üzerinden iletişim ve yardım al
- Yönetici paneli: ürün ekleme/düzenleme/silme + gelen siparişleri görme
- Tüm veriler `localStorage`'da saklanır (statik site)

## Kullanım

`index.html` dosyasını tarayıcıda aç.

## Dosyalar

- `index.html` — Ana sayfa ve tüm modallar
- `styles.css` — Tüm stiller
- `script.js` — Site mantığı (ürün, sepet, kullanıcı, sipariş, admin)
